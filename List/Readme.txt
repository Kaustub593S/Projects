Java To-Do List App

A simple Java console-based To-Do List application using an ArrayList and a custom Task class to manage tasks with priority and due date.

Features
- Add Tasks – Create a new task with name, priority, and due date.
- Remove Tasks – Delete a task by index or name.
- View Tasks – Display all tasks with their details.
- Sort Tasks – Sort tasks by priority or due date.
- Custom Task Class – Stores taskName, priority, and dueDate.

Tech Stack
- Language: Java
- Data Structure: ArrayList
- Concepts Used:
  - Classes & Objects
  - Encapsulation
  - Collections Framework
  - Comparators for sorting

How to Run
1. Clone the repository
   git clone https://github.com/your-username/todo-list-java.git
   cd todo-list-java

2. Compile the Java file
   javac Main.java

3. Run the program
   java Main
